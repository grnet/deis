User guide
==========

Client
-------

To install the deis client::

    $ curl -sSL http://deis.io/deis-cli/install.sh | sh
    $ sudo mv deis /usr/local/bin/deis


To login to the PaaS::

    $ deis login https://deis.paas.grnet.gr/
